First Class Trouble is an Asymmetrical 6 Player Multiplayer Game rooted in both Cooperation and Deception. Taking place on a Luxury Space Cruiser, 4 Residents, must survive an A.I. uprising, where 2 Players are secretly playing the role of Personoids. While the goal of the Residents is to Survive, Discover the Imposters, and Shut Down the Ship's A.I., the Personoids use a set of skills to sabotage ship systems, and Covertly Kill Residents.
Aside from benefits that cloud storage provides, like data that moves with the Account, Cloud Saving has been one of the most valuable tools when it comes to getting remote player data without needing a dedicated server.
With Cloud Saves, we are able to store:
  • Player Profile Info
  • Avatar Customization Info
    • Male and Female Outfit, including currently selected Gender
    • Vruumba Configuration ("Ghost Avatar")
  • Current Platform Info
    • Last Used Platform
    • Platform User Id
Utilizing Data Tables
One of the major components I developed for Customization was an easy-to-use set of Data Tables in C++ and Blueprint. This allowed for anyone to easily add new items to the Customization Menu. Each item includes variables like Mesh, Materials, Entitlement Code, Achievement Code, level requirement, and Quality.
  • Mesh and Materials are assigned to a Skeletal Mesh on the Preview Character, and In-Game Character
  • Codes and level requirement determines whether the item is usable
    • Achievement Codes are unlocked when a specific Achievement or Trophy has been Unlocked
    • Entitlement Codes are used for DLC redemption, twitch drops, and events. In this case, content won't be visible unless user owns Item
  • Quality separates items into different Color Categories, under a unique header
This is the Data used to construct an ItemCard used to select an Item in the Blueprint Menu 
USTRUCT(BlueprintType)
struct FPlayerCustomizationItemTableRow : public FCustomizationQualityItemTableRow
{
	GENERATED_BODY()

public:
	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	USkeletalMesh* AppearanceMesh;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	TArray<UMaterialInstance*> Materials;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	int32 UnlockLevel;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	int32 TokenCost;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	UTexture2D* IconThumbnail;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	UTexture2D* CharacterMask;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	FString AchievementCode;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	FString EntitlementCode;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	int32 DLCPackId;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	FText Description;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	bool DoMaskLegs;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	bool DoMaskFeet;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	bool VisibleWithoutOwning;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	TArray<UAkAudioEvent*> AKSounds;
};
Additionally, there's separate Data Tables for the Vruumba Avatar Customization 
USTRUCT(BlueprintType)
struct FVruumbaCustomizationTableRow : public FCustomizationQualityItemTableRow 
{
	GENERATED_BODY()

public:
	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	int32 UnlockLevel;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	int32 TokenCost;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	UTexture2D* IconThumbnail;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	FString AchievementCode;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	FString EntitlementCode;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	FText Description;
};

USTRUCT(BlueprintType)
struct FVruumbaSkeletalMeshCustomizationItemTableRow : public FVruumbaCustomizationTableRow
{
	GENERATED_BODY()

public:
	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	USkeletalMesh* Mesh;

	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	TArray<UMaterialInstance*> Materials;
};

USTRUCT(BlueprintType)
struct FVruumbaAnimatedMeshCustomizationItemTableRow : public FVruumbaSkeletalMeshCustomizationItemTableRow {
	GENERATED_BODY()

public:
	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	TSubclassOf<UObject> AnimationClass;
};

USTRUCT(BlueprintType)
struct FVruumbaParticlesCustomizationItemTableRow : public FVruumbaCustomizationTableRow {
	GENERATED_BODY()

public:
	UPROPERTY(EditAnywhere, BlueprintReadWrite)
	UParticleSystem* ParticleSystem;
};
